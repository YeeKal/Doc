## structure

- _layouts
  - default.html: 网页模板框架，包含_includes中组件
  - post.html: 文章模板, 由文章直接继承
  - page.html: 非文章页面模板
- index.html: 继承default
- about.md: 继承page
- category/index.html: 继承page
- archive/index.html: 继承page
- tag/index.html: 继承page
