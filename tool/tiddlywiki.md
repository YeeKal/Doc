## server

Installing TiddlyWiki on Node.js

```js
apt install nodejs
apt install npm

npm install -g tiddlywiki

tiddlywiki mynewwiki --init server  // to create a folder for a new wiki that includes server-related components
tiddlywiki mynewwiki --listen
```
